–°–æ–∑–¥–∞—Ç—å –ø–∞–ø–∫—É:  `os.mkdir(dirname)` –∏–ª–∏ `os.makedirs(dirname)`. `mkdir` —Å–æ–∑–¥–∞–µ—Ç –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—é –∏ –≤—ã–∑—ã–≤–∞–µ—Ç –æ—à–∏–±–∫—É, –µ—Å–ª–∏ –æ–Ω–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç. `makedirs` —Å–æ–∑–¥–∞–µ—Ç –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –∏ –≤—Å–µ –Ω–µ—Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –≤ —É–∫–∞–∑–∞–Ω–Ω–æ–º –ø—É—Ç–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –≤ `dirname` —É–∫–∞–∑–∞–Ω –ø—É—Ç—å `./dir1/dir2/dir3`, –∏ –ø—Ä–∏ —ç—Ç–æ–º dir1 –∏ dir2 –Ω–µ —Å—É—â–µ—Å—Ç–≤—É—é—Ç, —Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –∏—Ö —Å–æ–∑–¥–∞—Å—Ç).
    
–ü—Ä–æ–≤–µ—Ä–∏—Ç—å, —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –ª–∏ –ø–∞–ø–∫–∞: `os.path.exists(dirname)` (–≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True` –∏–ª–∏ `False`).
–ü—Ä–æ–π—Ç–∏—Å—å –ø–æ –∫–∞—Ç–∞–ª–æ–≥—É: `os.walk(dirname)`.

–í–æ—Ç —Ç–∞–∫–æ–π –º–∞–ª–µ–Ω—å–∫–∏–π —Å–Ω–∏–ø–ø–µ—Ç, –≤–æ–∑–º–æ–∂–Ω–æ, –≤–∞–º –ø—Ä–∏–≥–æ–¥–∏—Ç—Å—è:
```python
import os

if not os.path.exists(directory):
    os.makedirs(directory)
```
